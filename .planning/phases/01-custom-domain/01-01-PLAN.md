<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Configure GoDaddy DNS and Netlify custom domain</name>
  <files>No files — this is dashboard configuration in GoDaddy and Netlify</files>
  <action>
    This task requires manual configuration in two web dashboards. Claude cannot access these services.

    **Step A — Audit GoDaddy DNS zone (do this FIRST):**
    1. Log in to GoDaddy > My Products > DNS for no-bshomes.com
    2. Screenshot the current DNS records for reference
    3. Look for any existing A records on `@` — if GoDaddy has default/parking A records, DELETE them (they will conflict with the Netlify A record)
    4. Look for any existing CNAME on `www` — if present and not pointing to nobshomes.netlify.app, DELETE it
    5. Do NOT touch any MX records — these are needed for Phase 2 email setup

    **Step B — Add DNS records in GoDaddy:**
    1. Add A record: Host = `@`, Value = `75.2.60.5`, TTL = 1 hour (or lowest available)
    2. Add CNAME record: Host = `www`, Value = `nobshomes.netlify.app`, TTL = 1 hour (or lowest available)
    3. Save changes

    **Step C — Register custom domain in Netlify:**
    1. Go to Netlify site dashboard > Site configuration > Domain management
    2. Click "Add custom domain" > enter `no-bshomes.com`
    3. If Netlify prompts to "Change nameservers" or transfer DNS — DECLINE. Choose "Use external DNS" instead. This is a locked decision: do not transfer nameservers away from GoDaddy.
    4. Add `www.no-bshomes.com` as a domain alias
    5. Netlify will show "Awaiting external DNS" until propagation completes — this is expected

    **Step D — Trigger a deploy:**
    1. In Netlify, go to Deploys and trigger a manual deploy (or push any commit to master)
    2. This ensures the live site reflects the current codebase including the contact-data.ts values (CONT-01 and CONT-02)

    IMPORTANT: Do steps B and C at the same time (or C first) so Netlify is ready to provision the SSL certificate as soon as DNS propagates.
  </action>

  <done>GoDaddy has A record @ -> 75.2.60.5 and CNAME www -> nobshomes.netlify.app. Netlify has no-bshomes.com registered as custom domain. A deploy has been triggered.</done>
