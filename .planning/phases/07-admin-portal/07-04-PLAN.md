- Wrap editor in a border rounded-xl container

Create `src/app/api/admin/blog/route.ts`:
- GET: auth check, query all blog posts ordered by createdAt desc, return JSON. Include id, title, slug, excerpt, published, createdAt for list view.
- POST: auth check, read { title, content, excerpt, published } from body
  - Generate slug from title: lowercase, replace spaces with hyphens, remove non-alphanumeric (except hyphens), trim trailing hyphens
  - Insert into nbsBlogPosts with title, slug, content, excerpt, published
  - Return created record with 201

Create `src/app/api/admin/blog/[id]/route.ts`:
- GET: auth check, parse id, select by id, return post or 404
- PUT: auth check, parse id, read { title, content, excerpt, published } from body
  - Regenerate slug from title (keep same slug generation logic)
  - Update record with `db.update(nbsBlogPosts).set({ title, slug, content, excerpt, published, updatedAt: new Date() }).where(eq(nbsBlogPosts.id, id))`
  - Return updated record
- DELETE: auth check, parse id, delete, return 204
