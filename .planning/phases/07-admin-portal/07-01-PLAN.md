Install dependencies:
```bash
npm install drizzle-orm pg
npm install -D drizzle-kit @types/pg
```

Create `src/lib/db.ts`:
- Import drizzle from "drizzle-orm/node-postgres" and Pool from "pg"
- Create Pool with `connectionString: process.env.DATABASE_URL!`, `ssl: { rejectUnauthorized: false }`, `max: 3`, `idleTimeoutMillis: 10000`
- Export `db` using `drizzle({ client: pool })`

Create `src/lib/schema.ts` with three tables using pgTable from "drizzle-orm/pg-core":

**nbsBlogPosts** ("nbs_blog_posts"):
- id: serial PK
- title: text NOT NULL
- slug: text NOT NULL UNIQUE
- content: text NOT NULL (stores HTML from TipTap)
- excerpt: text NOT NULL DEFAULT ""
- published: boolean NOT NULL DEFAULT false
- createdAt: timestamp("created_at") defaultNow NOT NULL
- updatedAt: timestamp("updated_at") defaultNow NOT NULL
- Index on slug: "nbs_blog_posts_slug_idx"

**nbsTestimonials** ("nbs_testimonials"):
- id: serial PK
- name: text NOT NULL
- location: text NOT NULL DEFAULT ""
- quote: text NOT NULL
- rating: integer NOT NULL DEFAULT 5
- displayOrder: integer("display_order") NOT NULL DEFAULT 0
- active: boolean NOT NULL DEFAULT true
- createdAt: timestamp("created_at") defaultNow NOT NULL

**nbsGalleryImages** ("nbs_gallery_images"):
- id: serial PK
- blobUrl: text("blob_url") NOT NULL
- alt: text NOT NULL DEFAULT ""
- projectName: text("project_name") NOT NULL DEFAULT ""
- displayOrder: integer("display_order") NOT NULL DEFAULT 0
- createdAt: timestamp("created_at") defaultNow NOT NULL

Create `drizzle.config.ts`:
- schema: "./src/lib/schema.ts"
- out: "./drizzle"
- dialect: "postgresql"
- dbCredentials.url from DATABASE_URL env var
- tablesFilter: ["nbs_*"] — CRITICAL: only manage nbs_ tables, ignore HouseFinder tables

Export all table definitions from schema.ts for use by other modules.

Generate migration SQL:
```bash
npx drizzle-kit generate
```
This creates SQL in `drizzle/` directory. Review the generated SQL to confirm it only creates nbs_ tables and does NOT touch any existing tables.

Apply migration to the database:
```bash
npx drizzle-kit migrate
```
Do NOT use `drizzle-kit push` — it can cause data loss on a shared production database.
